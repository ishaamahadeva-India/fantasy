# **App Name**: Ultra-Posh

## Core Features:

- Multimedia Content Delivery: Serve high-quality articles, quizzes, audio narrations, and videos tailored to the user's interests.
- Daily News Quiz Generation: Use a generative AI tool to create a Daily News Quiz based on trusted sources.
- Personalized Knowledge Insights: Track and visualize user progress, highlighting growth areas and time invested in learning through elegant charts.
- Subscription Management: Offer a tiered subscription model (Free, Premium, Pro) with clear benefits, managed via in-app billing and Stripe.
- User Profile and Vault: Allow users to save interesting facts and figures to their own Knowledge Vault

## Style Guidelines:

- Primary color: Soft gold (#E6C87A) for CTAs and highlights, evoking luxury and intelligence.
- Background color: Deep charcoal (#0B0D12) for a serene and premium feel.
- Accent color: Calm blue (#6DAFFE) for links and informational elements, creating a subtle and harmonious contrast.
- Headline font: 'Poppins' sans-serif for headings with a precise, contemporary feel. Note: currently only Google Fonts are supported.
- Body font: 'Inter' sans-serif for body text, ensuring readability with a modern, neutral look. Note: currently only Google Fonts are supported.
- Quiz number font: 'JetBrains Mono' monospace for quiz numbers/stats with a technical look. Note: currently only Google Fonts are supported.
- Minimalist layout with generous whitespace, creating a clutter-free and elegant environment.
- Smooth transitions with fade and subtle vertical slide, enhancing micro-interactions and providing refined feedback.
