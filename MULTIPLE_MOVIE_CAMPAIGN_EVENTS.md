# Event Templates for Multiple Movie Campaigns

This document lists all available event templates that can be added to multiple movie campaigns.

## Total Events Available: 20

---

- **Rules**: The location of the main stage event will be considered final.

## How to Use These Events

1. **When creating a multiple movie campaign**, click "Add Event" button
2. Click "Select All Events" to add all 20 events at once
3. Or select individual events from the template list
4. Customize dates, points, and options as needed for your campaign
5. For multiple movie campaigns, you can assign events to specific movies or make them campaign-wide

---

## Notes for Multiple Movie Campaigns

- Events can be **campaign-wide** (applies to all movies) or **movie-specific** (assigned to a particular movie)
- When adding events, you can leave the "Movie" field blank for campaign-wide events
- Or select a specific movie from your campaign's movie list
- Points and difficulty levels can be adjusted per event
- Start and end dates should be set appropriately for each event
